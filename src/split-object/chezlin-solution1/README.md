# Split Object

Split an object into an array of objects. Each entry in the array will be one
key/value pair from the object.

---

## Strategy

Loop through object.
Key/value pairs.
preserve the nested keys when converting it to an array of objects.

---

## Implementation

The method used to execute the desired outcome was the "Object.entries" method.
The method preserve all key/value pairs while converting the object into an array.
Considering various object methods that could provide the desired outcome.
